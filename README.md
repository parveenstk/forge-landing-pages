# 🚀 Nuxt 4 App

A modern web app built with [Nuxt 4](https://nuxt.com) – powered by Vue 3, Vite, and Nitro.

## 🛠️ Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Start production server
npm run start

# Or generate static site
npm run generate
